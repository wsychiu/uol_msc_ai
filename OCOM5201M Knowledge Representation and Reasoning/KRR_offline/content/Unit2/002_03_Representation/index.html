    Lesson 3: Representation

    Learning outcomes for lesson 3

    After completing this lesson you should be able to:

    • use first order logic to represent knowledge about various situations.

    One of the challenges in using first order logic to represent knowledge, is that the knowledge is often described in natural language (English in our case) which can be ambiguous. We also see that sometimes exactly the same phrase in English has to be represented in the logic in quite different ways. This means that the process of writing logical expressions to represent knowledge is not a matter of translating English words into logic in a fixed way – you have to think about the meaning (sometimes the most likely of several meanings in the context).

    We will go through various examples. These examples involve geometric shapes and this type of example exists in various forms called "Tarski's world". You can experience an interactive version. Note that this may not run in all browsers and you may need to download the example to run it with java. Although some links on the web page do not work, the java program itself works correctly. To be clear: this is an additional resource and you do not need to use it to understand the examples below.

    Shapes scenarios

    We work with the following scenario, two examples of which are shown in the diagrams below. We are representing the arrangement of shapes in simple pictures, with the following constraints:

    • A picture can contain any number of shapes greater than zero.
    • Every shape in a picture is one of these kinds: a triangle, a circle, or a square.
    • There can be any possible mixture of kinds in a picture. For example, there can be shapes of every kind, or there might only be shapes of one kind.
    • two shapes may touch each other, but not overlap
    • one shape can be inside another, but when this happens nothing can be inside the innermost one
    • the relationships inside and touching can only be true of different things. Nothing is inside itself and nothing touches itself.

A smaller circle floats below and to the right-hand side of the triangle. To the right of the circle is the lower left-hand corner of a larger square which contains a smaller triangle in the lower left and a circle, the same size as the one previously described, in the upper right-hand corner.

    At first we are not going to represent these constraints themselves. We are assuming these constraints are given and then representing additional statements about pictures. We do this by going through a sequence of examples which get more complex as we go on.

    \[ \begin{array}{|llll|} \mathsf{Relation~ Symbol} &\mathsf{Arity} &\mathsf{Example} &\mathsf{Meaning~ of~ example}\\ \hline \mathit{circle} & 1 & \mathit{circle}(x) & x \mathsf{~ is~ a~ circle} \\ \mathit{triangle} & 1 & \mathit{triangle}(x) & x \mathsf{~ is~ a~ triangle} \\ \mathit{square} & 1 & \mathit{square}(x) & x \mathsf{~ is~ a~ square} \\ \mathit{touches} & 2 & \mathit{touches}(x,y) & x \mathsf{~ touches~ } y \\ \mathit{inside} & 2 & \mathit{inside}(x,y) & x \mathsf{~ is~ inside~ } y \\ \hline \end{array} \]

    We next take some formulas and take the domain of interpretation to be the set of all shapes in a picture. We look at whether the formulas are true in some or all of these interpretations given the constraints stated above.

    Note that being true in all these interpretations is not the same as being logically valid – being valid means being true in all possible interpretations including ones where something can be a triangle and a square at the same time.

    \(\def\cl{\mathit{circle}}\) \(\def\tl{\mathit{triangle}}\) \(\def\sr{\mathit{square}}\) \(\def\tch{\mathit{touches}}\) \(\def\ins{\mathit{inside}}\)

    Single quantifier examples about types of shape: universal case

    1. \(\forall x\, \cl(x)\)

      This means every shape is a circle. This is true in interpretations 1 and 2, and false in 3 and 4.

      1) A large circle is placed towards the upper left-hand corner of this rectangle. This circle contains another circle half its size inside it. To the right-hand side are two more circles, one directly above the other with a gap in between.

      3) This rectangle is identical to rectangle one, except it contains a triangle on the far right-hand side, adjacent but not touching the two separate circles.

      4) This rectangle contains a triangle in the lower left-hand corner. Another larger triangle is located above it, slightly to the right and pointing towards one o’clock. On the right-hand side is a large square which contains a smaller triangle positioned slightly right of centre within it. None of the shapes are touching.

      The formula \(\forall x\, \cl(x)\) is straightforward, but we can add a negation sign to it in two different ways which it is important to distinguish.

      • \(\neg \forall x\, \cl(x)\) will be true exactly when \(\forall x\, \cl(x)\) is false and vice versa. This formula is true in interpretations 3 and 4, and false in 1 and 2. The formula \(\neg \forall x\, \cl(x)\) means that not everything is a circle.

      • \(\forall x\, \neg \cl(x)\) which means no shape is a circle. This formula is true in interpretation 4, and false in 1, 2 and 3.

    2. \(\forall x\, (\tl(x) \vee \sr(x))\) and \((\forall x\, \tl(x)) \vee (\forall x\, \sr(x))\)

      These two formulas mean different things:

      • \(\forall x\, (\tl(x) \vee \sr(x))\) means each shape is a triangle or a square. It is true in examples 1 and 3 below, and false in 2 and 4.

      • \((\forall x\, \tl(x)) \vee (\forall x\, \sr(x))\) means that all of the shapes are triangles or all of the shapes are squares. This is true for example 3 and false in 1, 2, and 4 of the following interpretations.

      4) This rectangle contains five equally-sized circles in a random grouping. None of them are touching.

    3. \((\forall x\, \tl(x)) \to (\forall x\, \sr(x))\) and \(\forall x\, (\tl(x) \to \sr(x))\)

      Here you need to remember the definition of when a formula \(\alpha \to \beta\) is true. Such a formula is true whenever \(\alpha\) is false, and it is also true whenever \(\beta\) is true. The only case \(\alpha \to \beta\) is false is when \(\alpha\) is true and \(\beta\) is false.

      • \((\forall x\, \tl(x)) \to (\forall x\, \sr(x))\) is true in the interpretations 1, 3, and 4 above because \(\forall x\, \tl(x)\) is false in each of these. In interpretation 3 there is another reason why \((\forall x\, \tl(x)) \to (\forall x\, \sr(x))\) is true: because \(\forall x\, \sr(x)\) is true there. In interpretation 2 the formula \((\forall x\, \tl(x)) \to (\forall x\, \sr(x))\) is false because everything is a triangle but not everything is a square.

      • \(\forall x\, (\tl(x) \to \sr(x))\) is true in the interpretations 3 and 4, but false in both 1 and 2. In cases 3 and 4 the formula \(\tl(x)\) is false for every possible \(x\) in the domain, so \(\tl(x) \to \sr(x)\) is true for every element of the domain. In both 1 and 2 we can find an element of the domain for which \(\tl(x)\) is true but \(\sr(x)\) is false. In 1 there is only one such element; in 2 there are six.

    Single quantifier examples about types of shape: existential case

    1. \(\exists x\, \cl(x)\)

      This means at least one shape is a circle. This is true for interpretations 1, 2 and 3 below, and false in 4.

      The diagram contains four equal-sized rectangles labelled one to four in two rows, each of which contains an assortment of other shapes.

      1) A large circle is placed towards the upper left-hand corner of this rectangle. This circle contains another circle half its size inside it. To the right-hand side are two more circles, one directly above the other with a gap in between.

      2) This rectangle contains a solitary small circle towards its lower edge, slightly left of centre.

      3) This rectangle is identical to rectangle one, except it contains a triangle on the far right-hand side, adjacent but not touching the two separate circles.

      4) This rectangle contains a triangle in the lower left-hand corner. Another larger triangle is located above it, slightly to the right and pointing towards one o’clock. On the right-hand side is a large square which contains a smaller triangle positioned slightly right of centre within it. None of the shapes are touching.

      The formula \(\exists x\, \cl(x)\) is straightforward, but we can add a negation sign to it in two different ways which it is important to distinguish.

      • \(\neg \exists x\, \cl(x)\) will be true exactly when there is no circle in the picture. This formula is true in interpretation 4, and false in the others.

      • \(\exists x\, \neg \cl(x)\) which means some shape is not a circle. This formula is true in interpretations 3 and 4, and false in 1 and 2.

    2. \(\exists x\, (\tl(x) \wedge \sr(x))\) and \((\exists x\, \tl(x)) \wedge (\exists x\, \sr(x))\)

      • The formula \(\exists x\, (\tl(x) \wedge \sr(x))\) says that there is a shape which is both a triangle and a square. This will always be false for the interpretations we are considering.

      • \((\exists x\, \tl(x)) \wedge (\exists x\, \sr(x))\) means that there is a triangle and there is a square. This time there is no requirement that the triangle and the square have to be the same thing to make the formula true. In the interpretations shown below, this is true in 1, and false in the others.

    3. \((\exists x\, \tl(x)) \to (\exists x\, \sr(x))\) and \(\exists x\, (\tl(x) \to \sr(x))\)

      • The interaction between \(\exists\) and \(\to\) can be confusing. While it is not all logically wrong to write a formula of the shape \(\exists x \; (p(x) \to q(x))\), it is almost certainly not saying anything useful. The example here should help you understand what it does mean.

      • The formula \((\exists x\, \tl(x)) \to (\exists x\, \sr(x))\) captures the idea that if a picture has a triangle then it also has a square. This is true for pictures with no triangles at all, and it's true for any picture where there is both a square and a triangle. Otherwise it's false. This formula expresses a natural constraint where you can easily imagine the pictures for which it is true.

      • \(\exists x\, (\tl(x) \to \sr(x))\) looks similar to the other formula, but it is not nearly so obvious for which pictures it will be true. In the interpretations pictured below, this is true in 1, 2, and 3, but false in 4. In interpretation 4 the formula is false because it is impossible to find an element of the domain making \(\tl(x) \to \sr(x)\) true. In every case \(x\) has to be a triangle and not a square. In the interpretations 1, 2 and 3, there is at least one value for \(x\) where the formula is true. In interpretation 2 there is only one element making \(\tl(x)\) false; in 1 and 3 there are several elements which are not triangles.

    Nested combinations of quantifiers

    1. \(\exists x\, \exists y\, \exists z\, (\tl(x) \wedge \tl(y) \wedge \tch(x,z) \wedge \tch(y,z))\)

      • This might appear to say that there are two triangles that both touch the same third shape. However, there is no requirement that the \(x\), \(y\), and \(z\) are all different from each other. In first order logic it is possible that the same individual is referred to by two different variables. So the formula can be true where there is only one triangle, but that triangle touches some other shape. It is also possible that the formula is true because one triangle touches another triangle. In the interpretations 1 and 2 below the formula is true.

        Select for full text description

        This diagram contains two equal-sized rectangles labelled one and two, each of which contains an assortment of other shapes.

        1) In this rectangle there is a large square on the left-hand side with two triangles, one smaller than the other, inside it. The two triangles are not touching each other, and they are not touching the square they are inside. To the upper right there is a smaller square with a very small triangle inside it, not touching the square. In the lower right-hand corner of the rectangle there is a small triangle which is touching the lower left-hand corner of a larger square. This square contains a triangle, one corner of which is touching the square’s right-hand edge.

        2) On the left-hand side of this rectangle there is a large square which has five equally-sized triangles inside it. Towards the top edge are two triangles spaced apart. Below this is a triangle pointing down. This is touching another triangle pointing up on its slanted edge. Adjacent but not touching this on the right is another triangle pointing upwards not touching anything. To the right is a smaller square that contains one triangle. This is pointing downwards and touching the square on its bottom edge, slightly right of center.

      • To say there are two different triangles that both touch something, we need to make this explicit. One way is like this:
        \(\begin{array}{l} \;\\ \exists x\, \exists y\, \exists z\, \\ \quad(\tl(x) \wedge \tl(y) \wedge \tch(x,z) \wedge \tch(y,z) \wedge \neg (x = y)). \\ \;\\ \end{array}\)
        This formula is now only true in interpretation 1 above.

        If we need to say that \(x\), \(y\), and \(z\) are all different from each other, then this can be expressed by \(\neg (x = y) \wedge \neg (y = z) \wedge \neg (z = x)\).

    2. \(\exists x\, (\sr(x) \wedge \forall y\, (\tl(y) \to \ins(y,x)))\)

      This says that there is a particular square within which all the triangles can be found. Sometimes statements such as 'every triangle is in a square' can be ambiguous about whether there is a single square containing every triangle, or alternatively, each triangle is inside some square, but different triangles might be in different squares.
      When representing such situations it may be important to clarify what is meant, or to select the more likely meaning in the given context.

      If a newspaper headline reads 'There is a rat within 20 feet of everybody' it is most likely to mean that for each person in the country it is possible to find a rat within 20 feet of them. It is not likely to mean what it literally says: that one particular rat is so enormous as to be within 20 feet of everyone at once. In this example if you take the English statement 'There is a rat … ' and try to translate word-for-word into \(\exists x\,(\mathit{rat}(x) \wedge \ldots\), then the result is most unlikely to represent the intended meaning.

    3. \(\forall y\, (\tl(y) \to \exists x\, (\sr(x) \wedge \ins(y,x)))\)

      This says that every triangle is inside some square, but this need not be the same square for every triangle. It allows the possibility that all the triangles are in the same square, or that each triangle is inside a different square. In the following interpretations, this formula is true in 1, 2, and 4. In 4 it is true because there are no triangles.

    In representing situations in logic it is important to be careful about the difference between situations that might sound similar at first. For example, saying that every triangle in inside some square is different to saying that every square has some triangle inside it. This latter statement can be represented by \(\forall x\, (\sr(x) \to \exists y\, (\tl(y) \wedge \ins(y,x)))\).
