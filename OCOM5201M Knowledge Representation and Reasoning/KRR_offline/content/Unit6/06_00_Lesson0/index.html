    About Unit 6

    Learning Outcomes

    After completing this unit you should be able to:

    • Use more advanced Prolog constructs.
    • Build your own implementation to evaluate modal logic formulas.
    • See how tense logic works as a special case of modal logic.
    • See how Prolog terms can be manipulated in Prolog to check the syntax of formulas in your implementation.

    Overview

    This unit provides experience in writing Prolog and also in understanding some of the concepts around relations and modal logic. It should help develop your understanding of description logic and will introduce you to tense logic.

    The unit will guide you through a series of stages in which you are given a number of tasks. In order to help you do these tasks, some basic Prolog programming is introduced first.
